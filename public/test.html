Geo url scheme

  1. geo:37.786971,-122.399677 - show a point in San Francisco.
  2. geo:37.786971,-122.399677,1000 - show a point in San Francisco; altitude=1000 is ignored
  3. geo:37.786971,-122.399677;u=35 - show a point in San Francisco; u=35 is ignored
  4. geo:37.786971,-122.399677?z=17 - show a point in San Francisco with zoom=17
  5. geo:0,0?q=37.78918,-122.40335 - show a point in San Francisco
  6. geo:0,0?q=37.78918,-122.40335&z=16 - show a point in San Francisco with zoom=16
  7. geo:35.341714,33.32231(Custom%20Title) - show a point with 'Custom Title' in Kyrenia
  8. geo:35.3381607,33.3290564?q=Kyrenia%20Castle - search for 'Kyrenia Castle' in 35.3381607,33.3290564
  9. geo:0,0?q=Where%20to%20eat - search for 'Where to eat' in the current (last) location
  10. geo:0,0(Null%20Island) - show 0,0 point in Atlantic Ocean with 'Null Island' title
  11. geo:0,0?q=Restaurant - search for 'Restaurant' type around current position
  12. geo:0,0?q=87G7MXQ4%2BM5 - search for Plus Code '87G7MXQ4+M5' from current position (strings in Geo URI must be encoded so '+' is '%2B')
